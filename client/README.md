# POKECARDEX APP

This is a simple application that allows users to interact with the Pokemon Trading Card Game API(https://pokemontcg.io/)

## Why
This project exists to help users filter the the TCG API to find specific cards or discover new ones. 

## Who Are We
We are Team 2 from Cohort 39 at Wyncode! We are also Pokemon Fans!

# Want to clone and see for yourself? 
## How

### Set up

- `git clone` this repo
- `cd` into it.
- `yarn install`
- `cd client && yarn install`

#### Dependencies

You will need the following packages to run the program 
- `yarn add axios`
- `yarn add concurrently`
- `yarn add express`
- `yarn add react router dom`
